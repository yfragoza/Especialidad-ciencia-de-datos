# 📊 Especialidad en Ciencia de Datos

Este repositorio contiene el desarrollo de mi proceso formativo en el bootcamp de Ciencia de Datos, donde se abordan fundamentos de programación, análisis de datos y herramientas tecnológicas aplicadas al entorno profesional.

---

## 📁 Organización del repositorio

El contenido está dividido por módulos, siguiendo la estructura del bootcamp:

### 🔹 Módulo 0: Introducción al programa

Contiene información general del bootcamp, normativa y planificación.

### 🔹 Módulo 1: Perfil de especialidad

Enfocado en el rol profesional en TI, habilidades requeridas y contexto laboral.

### 🔹 Módulo 2: Python para análisis de datos

Incluye fundamentos del lenguaje Python, estructuras básicas y ejercicios prácticos.

### 🔹 Módulo 8: Deep Learning

Introducción a redes neuronales y conceptos clave de aprendizaje profundo.

### 🔹 Módulo 9: Big Data

Conceptos fundamentales de datos masivos y tecnologías asociadas.

### 🔹 Módulo 10: Portafolio

Desarrollo de trabajos y evidencias para presentación profesional.

### 🔹 Módulo 11: Empleabilidad

Preparación para el mundo laboral, incluyendo CV y entrevistas.
